# api_url_prediction
- Clone the repo
```
git clone https://github.com/CodeNinjaUsman/api_url_prediction
```
- Activate conda environment
- Install requirements
```
pip install -r requirements.txt
```
- Run Server
```
python manage.py runserver
```
- Open https://localhost:8000/
- Give the link as:
```
{"link":"Link goes here"}
```
